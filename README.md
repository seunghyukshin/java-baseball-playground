## 기능목록
- [ ] 컴퓨터 볼 생성
  - [ ] 1~9사이의 난수
  - [ ] 중복되지 않아야 함
- [ ] 사용자 볼 입력받기
  - [ ] 1~9사이의 숫자만 가능
  - [ ] 3자리가 되어야함(아닐 시 재입력)
  - [ ] 중복되지 않아야 함
- [ ] 점수 계산하기
  - [ ] 자리가 같고, 값이 같으면 "Strike"
  - [ ] 자리가 다르고, 값이 같으면 "Ball"
  - [ ] 값이 존재하지 않으면 "Nothing"
- [ ] 게임 종료 및 재시작
  - [ ] 3Strike시 1round종료
  - [ ] 1:restart, 2:end
